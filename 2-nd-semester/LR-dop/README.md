# Отчет по лабораторной работе: Сравнительный анализ производительности REST API фреймворков

## 1. Цель работы
Закрепить навыки нагрузочного тестирования веб-приложений, научиться самостоятельно сравнивать производительность различных фреймворков (синхронных и асинхронных), анализировать влияние блокирующих операций на пропускную способность, формулировать рекомендации по оптимизации, а также освоить применение LLM-ассистента GigaCode для генерации кода и тестовых сценариев с использованием техник zero-shot, one-shot и few-shot.

## 2. Выбранные фреймворки

| Тип | Фреймворк | Версия | Комментарий |
|-----|-----------|--------|-------------|
| Синхронный | Flask | 2.3.x | Классический WSGI-фреймворк |
| Асинхронный | Sanic | 23.6.x | Современный ASGI-фреймворк |
| Асинхронный | FastAPI | 0.104.x | Современный ASGI-фреймворк |

**Обоснование выбора:**
- Flask — наиболее популярный синхронный фреймворк, прост в реализации
- Sanic — один из самых быстрых асинхронных фреймворков
- FastAPI — современный асинхронный фреймворк с автоматической документацией
